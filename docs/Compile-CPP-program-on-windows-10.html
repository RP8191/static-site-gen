Build C++ programs on windows 10

  1. install cygwin

  2. create simple c++ file (single.cpp in this example) in any editor.

#include <iostream>
using std::count;

int main(){
    cout<<"hello !";
    return 0;   
}
  1. start cygwin command prompt. Type 
$ cd "C:"
  1. User cd to desired directory where the cpp source file resides. Type
$ g++ -o single.exe single.cpp

  1. This will create the output file single.exe

  2. To run the single.exe, type :

$ chmod a+x single
$ ./single
  1. Should display following message on the prompt.
hello !
